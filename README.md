 Calculator App

A simple and fully functional Android calculator application developed using Kotlin.  
The app provides basic arithmetic operations with a clean and user-friendly interface.

 Features
- Addition, Subtraction, Multiplication, and Division
- Continuous operations
- Error handling (e.g., division by zero)
- Simple and clean UI
-  Dark/Light theme

 Deliverables
- Fully functional calculator app
- Clean and user-friendly UI
- Working continuous operations and proper error handling
- Complete source code with README

 Tech Stack
- Kotlin
- Android Studio
- XML for UI design

 How to Run the Project
1. Clone this repository
2. Open the project in Android Studio
3. Build and run the app on an emulator or physical Android device
